# Twitter OAuth

## Introduction ##

This project reinvents oauth authentication for twitter. This can help me to understand OAuth 1.0a authentication flow.

## Usage ##

- Execute ```npm install```

- Fill in ```consumerKey``` and ```consumerSecret``` in index.js.

- Execute ```npm start```
